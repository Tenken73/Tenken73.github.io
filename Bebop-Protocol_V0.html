The Bebop Protocol

A data-driven exploration of anime trends, genre networks, and cultural impact using econometric modeling and graph theory

Project Overview

1000+ Anime Analyzed
15 Genre Clusters
50K+ User Reviews
3 Decades Covered

Key Findings

Genre Evolution

Action and drama genres dominate across decades, but psychological thrillers saw a 300% increase in popularity since 2010.

Studio Influence

Network analysis reveals that top studios (Bones, MAPPA, Ufotable) share production teams, creating interconnected quality clusters.

Sentiment Patterns

NLP on user reviews shows that "character development" and "animation quality" are the strongest predictors of high ratings.

Methodology

  • Data Collection: Scraped MyAnimeList API for 1000+ titles (2000-2024)
  • Cleaning: Python (pandas) for data preprocessing, handled missing values via imputation
  • Analysis: K-means clustering for genre grouping, sentiment analysis (VADER) on reviews
  • Network Science: Graph theory (NetworkX) to map studio/director relationships
  • Visualization: Chart.js for interactive charts, D3.js for network graphs
  • Tools: Python, R, SQL, JavaScript
